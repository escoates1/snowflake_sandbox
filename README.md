# snowflake_sandbox
A code base designed to help you get quickly set up with trial Snowflake accounts to experiment with features.

## Set up

1. Pip install uv.
2. In the terminal, once in the root directory, run 'uv init'.
3. Run 'uv venv'.
4. Activate the virtual environment - e.g. '.venv\Scripts\activate'.
5. Run 'uv sync' to install all dependencies from the lockfile.
